<br/>
A full-stack system that predicts loan approval outcomes and explains why — pairing a FastAPI/SHAP backend with an animated React/Vite dashboard for transparent, human-readable model decisions.
Aspect	Detail
Stack	Python, FastAPI, SHAP, scikit-learn, React, Vite
Scale	Single-model inference service with real-time explanation generation
Performance	Sub-second SHAP explanation rendering per prediction
Security	Input validation on all API endpoints
Impact	Converts opaque model output into transparent, auditable decisions
Repository	View Repo
This project explores how interpretability techniques like SHAP can be embedded directly into a decision-support UI, rather than bolted on as an afterthought — letting end users see which factors drove a loan decision, not just the outcome.
